# NumConv — Number System Converter

A clean, dark-themed number system converter that runs entirely in the browser. No backend, no database, no API keys needed.

## 🔢 Features

- Convert between **Binary (Base-2)**, **Octal (Base-8)**, **Decimal (Base-10)**, and **Hexadecimal (Base-16)**
- **Live conversion** as you type
- **Click to copy** any result
- **Bit visualizer** — see the binary representation as individual bits (8, 16, or 32-bit)
- Input validation with error messages
- Fully responsive design

## 🚀 Deploy on GitHub Pages

1. Fork or upload this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch → `/ (root)`
4. Your site will be live at `https://yourusername.github.io/repo-name`

## 🛠 Tech Stack

- No frameworks, no dependencies, no build step
